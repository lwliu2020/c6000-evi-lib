This method is used to set up PLL multiplier.

Thanh Nguyen
Parameters:
multiMultiplier, Crystal = 24MHz, multi=10 ==> DSP core freqency = 24*10/2 = 120MHz.
Returns:
Status of setting up.

STATUS_FINISHED,STATUS_FAILED. 

 // 

 UARTFreq=(CRYSTAL_FREQ*multi)/(POST_PLL_DIVIDER*2); //calculate uart freqency
